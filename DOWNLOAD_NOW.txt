## 🚀 ПОСЛЕ СКАЧИВАНИЯ — 3 КОМАНДЫ ДО ЗАПУСКА:


Откройте Terminal (⌘ + Пробел → "Terminal"):

```bash
# 1. Перейдите в папку проекта
cd ~/Downloads/fastmine-landing

# 2. Запустите автоматическую подготовку
bash prepare-deploy.sh

# 3. Следуйте инструкциям скрипта
```

Скрипт `prepare-deploy.sh` автоматически:
✅ Заменит файлы на production-версии (без figma:asset)
✅ Создаст папку для изображений
✅ Проверит что всё готово

## 📸 СКАЧАТЬ ИЗОБРАЖЕНИЯ (ДЕЛАЕТСЯ В CHROME!)


После подготовки проекта нужно скачать 29 изображений:

1. **Откройте Figma Make** в Chrome (оставьте вкладку открытой)
2. **Нажмите** ⌘ + ⌥ + J (откроется Console)
3. **В Figma Make** откройте файл `COPY_THIS_SCRIPT.txt`
4. **Скопируйте** длинную строку (от `const` до `;`)
5. **Вставьте** в Console: ⌘ + V
6. **Нажмите** Enter
7. **Подождите** 15-20 секунд
8. Все PNG скачаются в Downloads

Затем в Terminal:
```bash
cd ~/Downloads/fastmine-landing
mv ~/Downloads/*.png public/assets/
```
