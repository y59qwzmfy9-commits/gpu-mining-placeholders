# 🎨 ЦВЕТОВАЯ СХЕМА ЛЕНДИНГА FASTMINE

## Реализованная схема фонов

```
┌─────────────────────────────────────────┐
│  HERO                      [⚫ ТЕМНЫЙ]  │  Эмоциональный вход, призыв
├─────────────────────────────────────────┤
│  HOW IT WORKS              [⚪ СВЕТЛЫЙ] │  Текст, шаги - нужна читаемость
├─────────────────────────────────────────┤
│  WHY (Сравнение)           [⚪ СВЕТЛЫЙ] │  Таблица, много информации
├─────────────────────────────────────────┤
│  INFRASTRUCTURE            [⚫ ТЕМНЫЙ]  │  Фотогалерея оборудования
├─────────────────────────────────────────┤
│  FORMATS                   [⚫ ТЕМНЫЙ]  │  Визуальные карточки с фонами
├─────────────────────────────────────────┤
│  DOCUMENTS                 [⚪ СВЕТЛЫЙ] │  Документы, списки - читаемость
├─────────────────────────────────────────┤
│  GALLERY                   [⚫ ТЕМНЫЙ]  │  Фотогалерея (темный фон)
├─────────────────────────────────────────┤
│  FAQ                       [⚪ СВЕТЛЫЙ] │  Много текста, Q&A
├─────────────────────────────────────────┤
│  FINAL CTA                 [⚫ ТЕМНЫЙ]  │  Финальный призыв, форма
└─────────────────────────────────────────┘
```

## Соотношение: 55% темных / 45% светлых

---

## 💡 ЛОГИКА ВЫБОРА

### ⚫ ТЕМНЫЕ СЕКЦИИ (5 из 9)

**Используются для:**
- 🎬 **Эмоционального воздействия** → Hero, Final
- 📸 **Визуального контента** → Infrastructure, Gallery
- 🎨 **Дизайнерских элементов** → Formats (карточки с фонами)

**Почему темный работает:**
- ✅ Передает индустриальную, технологичную атмосферу
- ✅ Делает фото оборудования более выразительными
- ✅ Акцент на контент (карточки, кнопки светятся на темном)
- ✅ Премиальное восприятие
- ✅ Снижает усталость глаз (майнинг = ночная работа)

---

### ⚪ СВЕТЛЫЕ СЕКЦИИ (4 из 9)

**Используются для:**
- 📝 **Текстового контента** → How, Why, Docs, FAQ
- 📊 **Таблиц и сравнений** → Why
- 📄 **Документации** → Docs
- ❓ **Q&A разделов** → FAQ

**Почему светлый работает:**
- ✅ Максимальная читаемость длинных текстов
- ✅ Четкость для сравнительных таблиц
- ✅ Серьезность для юридических документов
- ✅ Визуальная пауза между темными блоками
- ✅ Лучшая контрастность для сложной информации

---

## 🎯 ПОЧЕМУ НЕ 50/50 ЧЕРЕДОВАНИЕ?

### ❌ Проблемы равномерного чередования:

1. **Потеря идентичности**
   - Майнинг/ГПУ = темная индустрия
   - Слишком много белого размывает промышленную эстетику

2. **Визуальная усталость**
   - Постоянные резкие переходы
   - Нет ритма и дыхания

3. **Нелогичная группировка**
   - Gallery на белом выглядит как офисный лендинг
   - Docs на темном сложно читать

---

## 📈 РЕЗУЛЬТАТ ТЕКУЩЕЙ СХЕМЫ

### ✅ Преимущества:

**1. Промышленная атмосфера сохранена**
- 55% темного создает нужное настроение
- Серьезность и технологичность

**2. Функциональность превыше формы**
- Светлый там, где много читать
- Темный там, где смотреть

**3. Визуальный ритм**
```
Темный → Светлый блок → Темный блок → Светлый → Темный → Светлый → Темный
   1   →    2 секции   →  2 секции   →    2 секции    →   1
```
- Есть группировка и дыхание
- Нет монотонности

**4. Лучшая конверсия**
- Hero (темный) → сильный первый экран
- Final (темный) → эффектный призыв
- FAQ (светлый) → легко читать перед конверсией

## 🎨 ТОНКИЕ ДЕТАЛИ

### Использованы разные оттенки темного:
- `bg: #040307` - основной фон
- `panel: #030303` - карточки на темном
- Создает глубину без цветовых переходов

### Использованы разные оттенки светлого:
- `lightBg: #f8f8f9` - основной светлый фон
- `lightPanel: #ffffff` - карточки на светлом
- Контрастность без резкости

### Границы адаптивные:
- Темные секции: `stroke: rgba(255,255,255,0.08)`
- Светлые секции: `lightStroke: rgba(0,0,0,0.08)`
- Визуальная связность
