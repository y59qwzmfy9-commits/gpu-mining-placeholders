# Замена фото по одному

В `src/assets/` лежат **плейсхолдеры** (маленькие серые квадраты). Сайт собирается и работает.

Чтобы вставить свои фото:
1. Замените файл в `src/assets/` на своё изображение с **тем же именем** (например, `hero.png`, `logo.png`).
2. Имена файлов должны остаться такими же: hero.png, logo.png, gpu.png, asic.png, delivery.png, commissioning.png, grid.png, fastmine.png, container-modules.png, personnel.png, aerial-view.png, maintenance.png, new-stations.png, monitoring.png, full-cycle.png, guarantees.png, asic-hosting.png, gpu-mining.png, weichai.png, gas-infra.png, technician.png, baudouin.png, baudouin-complex.png, gas-pipeline.png, service-maintenance.png, financial-model.png, supermaly-generator.png, gas-pipeline-field.png, engi-blue-generator.png.
3. После замены сделайте коммит и пуш — Vercel пересоберёт сайт.

Можно заменять по одному файлу и сразу пушить.
