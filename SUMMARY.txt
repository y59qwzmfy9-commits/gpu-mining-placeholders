## 🎯 ВАШ ПЛАН ДЕЙСТВИЙ:


1️⃣  **Откройте файл START_HERE.md** (или QUICKSTART.md)
    ↓
2️⃣  **Export → Download ZIP** из Figma Make
    ↓
3️⃣  **Откройте Terminal.app** и выполните:
    ```bash
    cd ~/Downloads/fastmine
    bash prepare-deploy.sh
    ```
    ↓
4️⃣  **Откройте Chrome** (не Terminal!) и скачайте изображения:
    • Зайдите на Figma Make
    • Нажмите ⌘ + ⌥ + J
    • Вставьте скрипт из COPY_THIS_SCRIPT.txt
    • Нажмите Enter
    ↓
5️⃣  **Переместите изображения** (в Terminal):
    ```bash
    mv ~/Downloads/*.png public/assets/
    ```
    ↓
6️⃣  **Деплой на Vercel**:
    ```bash
    npm install
    npm i -g vercel
    vercel
    ```
    ↓
7️⃣  **ГОТОВО!** 🎉
